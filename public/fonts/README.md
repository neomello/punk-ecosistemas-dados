# Fontes Blender Pro

Esta pasta contém os arquivos da fonte Blender Pro usada nos títulos do site.

## Arquivos

- `Blender-Pro-Medium.ttf` - Peso normal (400)
- `Blender-Pro-Bold.ttf` - Peso negrito (700)

## Uso

A fonte é aplicada automaticamente em todos os títulos (`h1`, `h2`, `h3`, etc.) através da classe Tailwind `font-display`.

Para usar manualmente em qualquer elemento:

```tsx
<h1 className="font-display">Título com Blender Pro</h1>
```
